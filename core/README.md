# Play Core

High-performance media playback and analysis framework for Timbre Play.

## Supported Containers and Codecs

----------------------------------------------------
| Containers | Detected Extensions | Codec Support |
----------------------------------------------------
| Ogg        | `.ogg`              | Vorbis, Opus  |
| FLAC       | `.flac`             | FLAC          |
| Wav        | `.wav`              | Wav           |
| MPEG-4     | `.mp4`, `.m4a`      | AAC, ALAC     |
----------------------------------------------------
